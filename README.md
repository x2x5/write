# 顶会写作技能

一个本地可用的网页工具，用于管理和复用写作 `skills` 卡片，支持拖拽排序、使用次数统计、替补/垃圾分区、快速新增与复制。

## 功能概览

- 主力卡片区：展示常用卡片，支持拖拽排序
- 替补卡片区：折叠面板，存放暂不使用卡片
- 垃圾卡片区：折叠面板，支持清空垃圾
- 一键复制：复制模板 + 输入内容
- 使用次数统计：每次“复制全部”且输入非空时计数 +1
- 按次数排序：点击按钮按使用次数降序排序一次
- 次数清零：一键清空所有卡片使用次数
- 新增卡片弹窗：
  - 元模板（折叠显示）
  - 需求输入与一键复制元模板
  - 粘贴 AI 返回内容（第一行标题，后续为 skills）自动建卡

## 数据存储与数据源

### 1) 卡片模板数据源

- 默认读取：`skills.md`
- 回退读取：`README.md`（当 `skills.md` 不可读时）
- 也支持手动选择本地 `skills.md`

建议：把卡片模板维护在 `skills.md`，`README.md` 专门用于项目说明。

### 2) 用户操作数据（浏览器本地）

以下内容存储在浏览器 `localStorage`：

- 主力/替补排序与归类
- 新增卡片
- 垃圾卡片
- 使用次数
- 局部编辑状态

注意：清除浏览器站点数据、无痕窗口结束、换浏览器/设备会导致这些本地数据丢失。

## 本地启动

在项目根目录运行：

```bash
python3 -m http.server 8080
```

浏览器打开：

- <http://localhost:8080>

## 目录说明

- `index.html`：页面结构
- `styles.css`：页面样式
- `app.js`：交互逻辑与状态管理
- `skills.md`：卡片模板数据源（推荐维护）
- `README.md`：项目说明（当前文件）

## 使用流程（推荐）

1. 点击右上角“新增卡片”
2. 在“需求”输入框写需求
3. 点击“复制元模板 + 需求”，粘贴到外部 AI
4. 将返回结果粘贴到 `Skills` 输入框
   - 第一行：标题（建议 4-5 字）
   - 第二行起：完整 skills 内容
5. 点击“创建卡片”
